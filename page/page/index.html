Acceuil

Bienvenue dans le jeux motus

Avant d'aller plus loin, nous allons vous donner les règles du jeux
Le but du jeux est de trouver le mot du jour. Vous aurez 5 essais
Le nombre de lettres que compose le mot est affiché, vous n'aurez pas le droit d'essayer un mot avec un nombre de lettres différent
A chaque essai, vous aurez un retour : vert si la lettre était bien placé, orange si elle était au mauvais endroit
Une fois le mot trouvé ou les 5 essais échoués, vous devrez attendre le lendemain pour jouer à nouveau
Bonne chance !
